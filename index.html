Welcome to a doodle!

This is a silly web thing, which hopefully looks entertaining.

Once it starts, you can click the main nodes to flip their polarity, and drag and drop them around.

On Desktop, you can also:

  • Right Click: Add or remove nodes
  • Space: Open or close this popup
  • Left / Right: Change presets

On Mobile, instead:

  • Long Press: Add or remove nodes
  • Double Tap: Open this popup
  • Swipe Left / Right: Change presets

To get started, click anywhere.
